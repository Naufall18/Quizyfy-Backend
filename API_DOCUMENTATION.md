## 🚀 Quick Start

### For Backend Developers
```bash
# Setup
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

### For Mobile Developers
- Read: DESIGN_TO_API_GUIDE.md
- Reference: API_ENDPOINTS_DETAIL.md
- Follow: MOBILE_VS_WEB_GUIDE.md

### For Web Developers (Future)
- Same as mobile developers
- Choose: React/Vue/Svelte framework
- Follow: MOBILE_VS_WEB_GUIDE.md
